주식 관리 웹 애플리케이션

이 프로젝트는 Flask와 Pandas를 사용하여 주식 포트폴리오를 관리하는 간단한 웹 애플리케이션입니다.

주요 기능

  • 주식 목록 조회
  • 회사명 또는 종목코드로 검색
  • 열을 기준으로 정렬
  • 페이지네이션
  • 신규 주식 추가
  • 기존 주식 정보 수정
  • 등록된 주식 삭제

기술 스택

  • 백엔드: Python, Flask
  • 데이터 처리: Pandas
  • 데이터 저장소: Excel

참고: 이 파일은 프로젝트를 설명하는 정적 HTML 페이지입니다. 실제 애플리케이션을 실행하려면 `python app.py`를 실행하고 http://127.0.0.1:8000에 접속하세요.
